Movuca

  • Social CMS Engine
  • Demo: http://movu.ca/demo
  • Powered by web2py Python web framework

Requirements

  • Python 2.7+
  • pycrypto
  • PIL
  • web2py

Get started

  • Open your terminal

1. Get web2py using git (or download and unzip it)

  • git clone git://github.com/mdipierro/web2py.git web2py

2. Jump to applications folder

  • cd web2py/applications

3. Get Movuca from git (or download and unzip it)

  • git clone git://github.com/rochacbruno/Movuca.git demo

4. Start web2py development server

  • cd ..
  • python web2py.py -a "yourdesiredpassword"

5. Run Movuca installation process

  • Go to browser and run
  • http://localhost:8000/demo/setup/install

You have to see "bootstrap" on the screen

6. Enjoy

  • - http://localhost:8000/demo
